# better-windows-toolbar
Это тулбар винды, но красивее и с открытым исходным кодом!

# как использовать?

- скачиваем exe файл
- сохраняем куда-нибудь
- создаём папки для нужных медиа файлов и иконок (естественно кидаем их туда)
- создаём ссылку на скачанный exe файл (шорткат) и в поле "target" (или же цель) добавляем рядом с путём программы путь нужной папки (не забудте кавычки)
